# HTML5_QR_Code_scanning_for_phone_camera
## 說明
以前端方式達成掃描電子發票並把發票條碼輸入至 input 中的功能  

## 使用套件
[schmich/instascan](https://github.com/schmich/instascan)：啟動相機並掃描二維條碼  
[material](https://material.io)：提供前端頁面美化、排版等函式庫
